# POO e Tratamento de Exceções em Java

Os projetos desse repositório são voltados à aplicação de boas práticas de programação dentro de **Programação Orientada a Objetos (POO)** e estruturação de regras de negócio com **Tratamento de Exceções**. 

## Principais Conceitos Aplicados
* Orientação a Objetos, aplicando Herança, Polimorfismo, Encapsulamento, Abstração;
* Tratamento de Erros com blocos *try*, *catch*, *finally*, *throw* e *throws*;
* Criação de Exceções Customizadas;
* Uso de *Enum* e *BigDecimal* para maior precisão;
* Estruturas de Dados como *ArrayList*.

---

## Sobre os Projetos 

O projeto possui três sistemas distintos, cada um focado em resolver problemas do mundo real por meio da aplicação e prática dos conhecimentos adquiridos dentro e fora do meio acadêmico. 

### 1. Sistema de Venda de Ingressos
Simula o back-end de uma bilheteria de eventos. Foco total em validação de regras de negócio.
O sistema impede a venda e lança exceções personalizadas caso o cliente seja menor de idade, o setor esteja esgotado ou a meia-entrada seja solicitada sem documentação.

### 2. Sistema de Operações Bancárias
Simula transações financeiras e rendimentos de contas Corrente, Poupança e Investimento. Aplicando a classe *BigDecimal* para garantir precisão matemática, herança de classes para evitar repetição de código e aplicação de taxas específicas por tipo de conta.

### 3. Sistema de Folha de Pagamento
Simula o funcionamento de folha de pagamento de uma empresa de acordo com a modalidade de serviço assalariado, comissionado e horista. Nesse projeto, além dos princípios de herança e encapsulamento de dados, tem-se a aplicação de classe abstrata e conexão entre classes, bem como a utilização da estrutura de ArrayList.
